# prototype2-fix
Prototype2 FR Overwrite

Yo! J'update le guide, fonctionnel je viens de le tester à l'instant :)

Ce que nous avons besoin :

[list]
    [*]Le Jeu, installé en Russe si possible.
    [*]Le mod qui fix le lancement du jeu : https://www.nexusmods.com/prototype2/mods/42
    [*]Patch FR (si vous voulez, sans oublier les fichiers pour la version russe) : https://traductionjeux.com/Prototype2-Patch-FR.html
    [*].bat pour le lancement du jeu qui forcera le langage dans le registre sur 70, le problème étant que si le jeu est lancé en mode administrateur, il va overwrite sa valeur de langage en 82 (donc anglais) à chaque fois, et le patch FR ne marchera pas!
[/list]

1) Installer le jeu,
2) Télécharger puis extraire le mod dans le dossier racine
3) Télécharger, extraire (les 2 dossiers simultanément) où bon vous semble et déplacer son intégralité dans le dossier racine du jeu
4) Lancer le launch.bat pour pouvoir profiter pleinement du jeu! (https://github.com/KurenoDev/prototype2-fix)
